# Hi, I'm Sridhar 👋

### Electronics & Computer Engineer | Embedded Systems | IoT | FPGA

I'm an electronics and computer engineering professional based in **Ilmenau, Germany**, passionate about building things at the intersection of hardware and software — from low-level RTL design to full IoT systems.

## 🔧 What I Work On

- **Processor Architecture** — RISC CPU design with out-of-order execution, register renaming, and ROB in Verilog/SystemVerilog
- **FPGA Development** — Hardware synthesis and implementation on Altera Cyclone IV
- **Embedded Systems** — Firmware for ESP32, nRF52840, LoRa/LoRaWAN nodes
- **IoT & Home Automation** — Zigbee2MQTT, Node-RED, InfluxDB, Grafana, Docker
- **PCB Design** — KiCad schematics and layouts for custom RF/sensor boards
